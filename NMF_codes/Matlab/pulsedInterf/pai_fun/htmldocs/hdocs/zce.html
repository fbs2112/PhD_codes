PURPOSE

Zero-crossing Frequency Estimation

DESCRIPTION 

 Zero-crossing Frequency Estimation

 Estimates the instantaneous frequency of the input signal using the
 zero-crossing instantaneous frequency estimation algorithm.


 Usage:

     ife = zce( signal, window_length )

 Parameters:


    signal

      Input one dimensional signal to be analysed.

    window_length

      Moving window to count the average number of zero crossings each
      time it moves.
